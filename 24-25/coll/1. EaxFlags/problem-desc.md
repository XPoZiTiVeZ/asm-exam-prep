# Задача EaxFlags: Определение значения регистра EAX и флагов

Секция `.data` была размещена в памяти, начиная с базового адреса `0x4713a13c`. Укажите значение регистра `EAX` и флагов после выполнения указанных команд секции `.text`.

```asm
section .data
    a dw 0x2022, 0x2023, 0x2024, 0x2025
    s db "output", 0
    b dd 0xAB31FD32
    p dd b
    
section .text
    movzx eax, word[b + 2] ; (1)
    sub al, byte[p + 1]    ; (2)
    movsx ax, byte[a + 1]  ; (3)
```

Скопируйте 3 строки ниже в поле ответа и заполните прочерки значениями регистра EAX (в шестнадцатеричном виде) и флагов (0/1) после выполнения соответствующих инструкций:

(1) EAX = -  
(2) EAX = -, CF = -, OF = -, ZF = -, SF = -  
(3) EAX = -  

## Пример возможного формата ответа:

(1) EAX = 0xcafebabe  
(2) EAX = 0x00000000, CF = 0, OF = 0, ZF = 1, SF = 0  
(3) EAX = 0x00cafeba
