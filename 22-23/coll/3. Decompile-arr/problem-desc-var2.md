# Задача Decompile: Декомпиляция программы с массивом

Даны два фрагмента кода на языке Си с пятью пропусками:

```c
// объявление глобальных переменных
пропуск_1 a;
int A[23][пропуск_2][2];
int x, y, z;

// выражение-оператор
A[z][пропуск_3][пропуск_4] = пропуск_5;
```

Для этих фрагментов компилятор построил следующий код:

```asm
    imul eax, dword[z], 12
    add eax, dword[x]
    mov edx, dword[y]
    lea eax, [edx+eax*2]
    mov edx, dword[a]
    movsx edx, word[edx]
    mov dword[A+eax*4], edx
```

Вам необходимо восстановить пропуски в исходном фрагменте кода на языке Си. Выпишите их в ответе по одному на строке. В ответе должно быть ровно пять строк. Если не получается заполнить какой-то из пропусков, оставьте эту строку пустой.

На месте пропуска может быть либо число (в таком случае выписывайте число в десятичной системе счисления), либо одно из выражений, указанных ниже под буквами (в таком случае выписывайте соответствующую букву):

```text
A. char
B. short
C. int
D. char *
E. short *
F. int *
G. a
H. A
I. x
J. y
K. z
L. *a
M. *A
N. *x
O. *y
P. *z
Q. x + y
R. x + z
S. y + z
T. x * 9
U. y * 4
V. z * 2 
```

## Пример форматированного ответа:

```text
100
A
200
B
50
```
