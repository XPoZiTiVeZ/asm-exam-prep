Задача EaxFlags: Определение значения регистра ЕАХ и флагов

Секция `.data` была размещена в памяти, начиная с базового адреса `0x0804ea5e`. Укажите значение регистра `ЕАХ` и флагов после выполнения указанных команд секции `.text`.

```asm
section .data
    a dw 0х2020, 0х2021, 0x2022
    s db "march", 0
    b dd 0xD0CF11E0
    р dd b

section .text
    movsx еах, word[b + 2] ; (1)
    sub аl, byte[p]        ; (2)
    movzx ах, byte[a + 4]  ; (3)
```

Скопируйте З строки ниже в поле ответа и заполните прочерки значениями регистра `EAX` (в шестнадцатеричном виде) и флагов (0/1) после выполнения соответствующих инструкций:

```text
(1) ЕАХ = _
(2) ЕАХ = _, CF = _, OF = _, ZF = _, SF = _
(3) ЕАХ = _
```

Пример возможного формата ответа:

```text
(1) ЕАХ = _
(2) ЕАХ = _, CF = _, OF = _, ZF = _, SF = _
(3) ЕАХ = _
```
