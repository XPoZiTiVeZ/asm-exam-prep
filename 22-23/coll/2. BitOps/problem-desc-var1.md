# Задача BitOps: Битовые операции

На вход программе подается беззнаковое 32-битное целое число `N`. Требуется вывести число `M`, полученное из числа `N` следующим образом:
- Если есть непрерывная последовательность единичных битов в конце числа н, то эти биты становятся нулевыми.
- Все остальные биты заполняются единицами.
Если `N = O`, то число `M` состоит только из единиц.

## Примеры:

**Входные данные:**
```text
1536
```

**Результат работы**
```text
4294966783
```

**Входные данные:**
```text
1
```

**Результат работы**
```text
4294967294
```

## Ответ
<details>
<summary>Показать</summary>

```asm
extern io_get_udec
extern io_print_udec

section .text
global main
main:
    call io_get_udec
    mov ebx, eax
    dec ebx
    not eax
    or eax, ebx
    call io_print_udec    

    xor eax, eax
    ret
```
</details>
